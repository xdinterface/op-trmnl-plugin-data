# One Piece Birthday - TRMNL Plugin Data

Public data repository for the [One Piece Birthday TRMNL plugin](https://usetrmnl.com).

## Contents

- `data/` - Character birthday data (JSON)
- `images/` - Character images optimized for e-ink display

## Usage

This repo is polled by TRMNL devices to display One Piece character birthdays.

**Polling URL:** `https://raw.githubusercontent.com/xdinterface/op-trmnl-plugin-data/main/data/current.json`

## Data

- 937 characters with birthday data
- Includes name, affiliation, bounty, devil fruit, origin, and more
- Short bios scraped from One Piece Wiki
